PoolCensus Desktop App Instructions

Run a quick scan of public mining pool endpoints and view a local HTML report. Works on Windows, macOS, and Linux.

Basic usage

Double-click the PoolCensus Desktop App to start scanning.

While it runs, you’ll see a progress indicator.

When it finishes, it creates report.html in the same folder. Open that file to view the results.

Tip: If it takes a while, just leave it open until the scan completes.

Advanced (optional)

If you want to run from source: install Go from https://go.dev/dl, then run go run . from the repository root.

  • -scans 5 — do more scan passes
  • -verbose — show connection errors (otherwise you only see the progress bar)
  • pools.json next to the app — override the built-in pool list
