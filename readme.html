PoolCensus Desktop App

Run go run ./desktop-app (or build the binary to double-click it) to scan every endpoint listed inside desktop-app/pools.json. By default the Desktop App probes each server three times; adjust that with -scans 5 to control how many full passes are performed before rendering report.html.

A live spinner-style progress bar shows how far the scan has progressed. If one or more servers fail, rerun with -verbose to surface raw connection errors in the terminal.

Custom pools: place your own pools.json next to the desktop-app executable (the same directory the binary lives in). The app will automatically detect that copy and use it instead of the embedded list, so you can manage which pools are scanned without rebuilding.
